📄 Resume Classification Project This project is an NLP-based Resume Classification system that categorizes resumes into different job roles using Machine Learning.

🚀 Features Extracts text from resumes Performs text preprocessing (cleaning, tokenization, etc.) Uses TF-IDF vectorization Trained using Machine Learning model (Random Forest) Predicts job category from resume text Simple web app using Flask

🧠 Tech Stack Python Scikit-learn NLP (NLTK / SpaCy) Flask
